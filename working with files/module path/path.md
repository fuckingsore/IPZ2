## Модуль Path

### Визначення та використання
------------------------------

Модуль `path` пропонує утиліти для роботи з шляхами до файлів та каталогів.

### Синтаксис
-------------

Синтаксис включення модуля `path` у вашу програму:

```js
var path = require('path');
```
